PERFORMANCE METRICS EXPLAINED
??????????????????????????????????????????????????????????????????????????????????

PRODUCTIVITY SCORE (0-100)
  Formula: (Completion Rate × 50%) + (Rating Score × 50%)
  
  Example 1: Employee with 90% completion and 5-star rating
    Score = (90 × 0.5) + (100 × 0.5) = 45 + 50 = 95
  
  Example 2: Employee with 60% completion and 3-star rating
    Score = (60 × 0.5) + (60 × 0.5) = 30 + 30 = 60
  
  Interpretation:
    90-100 = Excellent performer
    80-89  = Good performer
    70-79  = Average performer
    60-69  = Below average
    <60    = Poor performer

TASK COMPLETION RATE (0-100%)
  Formula: Completed / (Completed + Pending) × 100
  
  Example: 18 completed, 2 pending
    Rate = 18 / 20 × 100 = 90%

PERFORMANCE RATING (1-5 STARS)
  1 Star = Poor         (Very poor, needs improvement)
  2 Stars = Below Avg   (Below expectations, training needed)
  3 Stars = Average     (Meets expectations)
  4 Stars = Good        (Exceeds expectations)
  5 Stars = Excellent   (Outstanding performance)

PERFORMANCE STATUS (DERIVED)
  Excellent = 5-star rating (or high score)
  Good      = 4-star rating
  Average   = 3-star rating
  Below Avg = 2-star rating
  Poor      = 1-star rating

SUPPORT & TROUBLESHOOTING
??????????????????????????????????????????????????????????????????????????????????

Issue: Charts not displaying
Solution:
  1. Check Chart.js CDN is loaded (browser console)
  2. Verify data is passed to view
  3. Check browser JavaScript errors (F12)
  4. Refresh page and try again

Issue: Employee doesn't appear after recording
Solution:
  1. Refresh the page
  2. Verify employee exists in system
  3. Check employee is assigned to department
  4. Check database for record

Issue: Incorrect productivity score
Solution:
  1. Verify calculation formula matches requirement
  2. Check decimal/rounding in calculation
  3. Review test case manually
  4. Check CalculateProductivityScore() method

Issue: Migration fails
Solution:
  1. Check SQL Server is running
  2. Verify connection string
  3. Check for existing columns
  4. Review migration errors in detail
  5. Try: dotnet ef database update --verbose

Issue: Page won't load
Solution:
  1. Check authorization (logged in as Admin)
  2. Verify no compilation errors
  3. Check application logs
  4. Restart application
  5. Clear browser cache
